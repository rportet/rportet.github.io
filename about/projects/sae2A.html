Résumé

Ce projet SAE 2A, réalisé en 175 heures, m'a permis de mettre en œuvre diverses compétences essentielles : C1 réaliser un développement d’application en PHP et Kotlin, C2 optimiser les performances, C3 administrer des systèmes informatiques communicants complexes, C4 gérer des données de l’information, C5 conduire un projet de bout en bout, et C6 collaborer au sein d’une équipe informatique. L’ensemble du projet s'articule autour de l'analyse, du développement d'une application web et mobile, la création d'un service web et la gestion d'une base de données.

Illustrations & Schémas

Déroulement du Projet

Analyse

Étude des besoins, élaboration des cas d'utilisation, réalisation de maquettes et définition des user stories pour déterminer les fonctionnalités clés.

Application Web

Développement d'une application web en PHP, dotée d'une interface responsive et d'une API RESTful pour la communication avec l'application mobile.

Application Mobile

Conception et développement d'une application mobile en Kotlin pour Android, connectée à l'API pour offrir une expérience utilisateur optimale.

Service Web

Mise en place d'un service web/ API RESTful facilitant l'échange de données entre l'application web et mobile via des requêtes HTTP.

Base de données et ORM

Conception de la base de données, définition des tables et relations, et utilisation d'un ORM pour faciliter l'accès et la manipulation des données.

Déploiement

Déploiement de l'application web, du service web et de la base de données via Docker, assurant une mise en production efficace et automatisée.
